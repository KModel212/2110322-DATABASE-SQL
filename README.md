# 🗄️ SQL Exercises

This repository contains SQL exercises that I worked on as part of my Database course.

The goal of these exercises is to practice writing queries, understanding relational databases, and working with real-world data scenarios.

## 📚 Topics Covered
- Basic SQL queries (SELECT, WHERE, ORDER BY)
- Joins (INNER JOIN, LEFT JOIN)
- Aggregations (COUNT, SUM, AVG)
- Grouping (GROUP BY, HAVING)
- Subqueries

## 🎯 Purpose
To strengthen my understanding of database concepts and improve problem-solving skills using SQL.
